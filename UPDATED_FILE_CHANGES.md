# Latest UI Updates

- Added **Finish Consultation** action card in OP Queue Management.
- Clicking **Finish Consultation** changes the selected patient/token status to **Completed**.
- Queue summary and queue tab counts now update from current queue data instead of fixed values.
- Queue action cards now use responsive sizing so all cards stay visible.
- Removed **Appointment In** and **Appointment Out** from Registration Desk navigation.
- Frontend build verified with `PUPPETEER_SKIP_DOWNLOAD=true npm install` and `npm run build`.
